# دليل فعاليات الجامعة الافتراضية

مشروع موقع إلكتروني لعرض وإدارة الفعاليات باستخدام PHP وMySQL وBootstrap.

## التشغيل المحلي

1. تأكد من تشغيل MySQL (تمت الإعداد لاستخدام XAMPP المحلي).
2. شغّل الخادم:
   ```bash
   php -S 127.0.0.1:8000
   ```
3. افتح المتصفح على:
   - http://127.0.0.1:8000/index.php
   - http://127.0.0.1:8000/admin/login.php

## بيانات الدخول للوحة التحكم

- اسم المستخدم: admin
- كلمة المرور: admin123
